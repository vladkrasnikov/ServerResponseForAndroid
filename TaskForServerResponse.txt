"Write a program that checks whether a given website is available (HTTP GET request returns response 200).
Checking is performed every 5 minutes using Broadcast Receiver (BR) and Service (S). BR launches S.
S executes a network request and stores the result in the database. S launches intent for BR, etc. "
"Results of checks are deduced from a DB on foliage in the main activity.
Additional activity displays a pie chart of percentage availability (available - not available) "


Ukrainian:
"Написати програму яка перевіряє чи доступний заданий вебсайт (запрос HTTP GET повертає респонс 200).
Перевірка виконується кожні 5 хвилин з допомогою Broadcast Receiver (BR) та Service (S). BR запускає S.
S виконує мережевий запит і зберігає результат в БД. S запускає інтент для BR и т д."
"Результати перевірок виводяться з БД на листвью в головне активіті.
Додаткове активіті виводить кругову діаграму доступності сату в процентах (доступний - недоступний)"
